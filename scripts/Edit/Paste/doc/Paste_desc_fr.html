La commande coller insère les entités qui ont été précédemment copiées dans le presse-papiers. La fonction Coller est particulièrement utile pour transférer des entités d'un dessin à un autre.
Tous les calques utilisés par le contenu du presse-papiers sont collés dans la liste des calques du dessin en cours. Les couches existantes portant le même nom peuvent être écrasées si désiré (option"écraser les couches" dans la barre d'outils des options).
Les références de bloc qui se trouvent dans le presse-papiers seront collées avec les définitions de bloc auxquelles elles se réfèrent. Les blocs du dessin courant peuvent être écrasés si désiré (option"écraser les blocs" dans la barre d'outils des options).
La barre d'outils offre également des outils pour mettre à l'échelle, faire pivoter ou retourner le contenu du presse-papiers pendant le collage.
