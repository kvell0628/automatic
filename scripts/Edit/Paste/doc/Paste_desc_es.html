El comando de pegar inserta las entidades que fueron previamente copiadas al portapapeles. Pegar es especialmente útil para transferir entidades de un dibujo a otro.
Todas las capas utilizadas por el contenido del portapapeles se pegan en la lista de capas del dibujo actual. Las capas existentes con el mismo nombre se pueden sobrescribir si se desea (opción"sobrescribir capas" en la barra de herramientas de opciones).
Las referencias de bloque que se encuentran en el portapapeles se pegarán junto con las definiciones de bloque a las que se refieren. Los bloques del dibujo actual se pueden sobrescribir si se desea (opción"sobrescribir bloques" en la barra de herramientas de opciones).
La barra de herramientas de opciones también ofrece algunas herramientas para escalar, rotar o voltear el contenido del portapapeles mientras se pega.
