1. Utilisez les outils de sélection pour préparer une sélection d'entités à copier dans le presse-papiers.
  2. Sélectionnez l'outil de copie.

L'outil de copie standard de QCAD agit comme l'outil de copie d'autres applications. Il ne vous permet pas de choisir un point de référence pour votre sélection mais calcule automatiquement le point de référence au centre de la sélection. Ce n'est pas toujours souhaitable car vous voulez probablement placer les entités collées à une position définie avec précision. Pour cela, QCAD propose des outils de copie alternatifs, plus spécifiques à la CAO : Copier avec référence.

La sélection est maintenant dans le presse-papiers interne de QCAD et peut être collée dans le même dessin ou dans un dessin différent qui est ouvert avec la même instance de QCAD. Notez que si vous quittez l'application QCAD, le contenu du presse-papiers est perdu. Le presse-papiers QCAD n'est accessible que par QCAD lui-même. Vous ne pouvez pas échanger des entités de dessin avec d'autres applications par copier-coller. Veuillez utiliser la fonctionnalité d'exportation et d'importation de QCAD pour cela.
