Las librerías de partes de QCAD son colecciones de símbolos y otros archivos de dibujo que se pueden utilizar en sus dibujos. Para insertar un elemento de una biblioteca de piezas, primero debe mostrar el navegador de la biblioteca utilizando esta herramienta.
Las piezas se pueden insertar en el dibujo mediante arrastrar y soltar. Una vez que el cursor del ratón está dentro del área de dibujo, puede utilizar las herramientas de ajuste habituales para posicionar la pieza insertada y la barra de herramientas de opciones para escalar, voltear o girar la pieza.
