1. Sélectionnez les entités que vous voulez faire tourner et contre-rotation.
  2. Lancez cet outil.
  3. Définissez le centre de la rotation principale avec la souris ou entrez une coordonnée dans la ligne de commande.
  4. Définissez le centre de la rotation pour les objets individuels. Ce deuxième centre de rotation est tourné avec les entités autour du premier centre.
  5. La boîte de dialogue de rotation deux s'affiche, dans laquelle vous pouvez entrer l'angle de rotation et l'angle de contre-rotation.
    Pour supprimer les entités originales, cochez "Supprimer l'original", pour les copier, choisissez "Conserver l'original". Vous pouvez également créer n'importe quel nombre de copies en choisissant "Copies multiples".
    Les nouvelles entités sont placées sur le même calque que les originaux et ont les mêmes attributs. Pour utiliser le calque courant et les attributs courants, cochez "Utiliser le calque courant et les attributs".
  6. Cliquez sur "OK" pour faire pivoter les entités.
