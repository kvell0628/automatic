1. Sélectionnez les entités que vous voulez faire pivoter.
  2. Lancez cet outil.
  3. Définissez le centre de la rotation à l'aide de la souris ou entrez une coordonnée dans la ligne de commande.
  4. La boîte de dialogue de rotation s'affiche, dans laquelle vous pouvez entrer le nombre de copies et l'angle de rotation.
    Si vous voulez spécifier l'angle de rotation avec le curseur de la souris, cochez le bouton du curseur de la souris qui est affiché à côté de l'entrée d'angle :

    Pour supprimer les entités originales, cochez "Supprimer l'original", pour les copier, choisissez "Conserver l'original". Vous pouvez également créer n'importe quel nombre de copies en choisissant "Copies multiples".
    Les nouvelles entités sont placées sur le même calque que les originaux et ont les mêmes attributs. Pour utiliser le calque courant et les attributs courants, cochez "Utiliser le calque courant et les attributs".
  5. Cliquez sur "OK".
  6. Si vous avez déjà choisi de spécifier l'angle de rotation avec la souris, vous devez d'abord spécifier un point de référence pour la rotation et ensuite un point cible. L'angle de rotation est l'angle délimité par le point de référence, le centre de rotation et le point cible.
