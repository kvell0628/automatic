1. Sélectionnez les entités que vous voulez mettre à l'échelle.
  2. Lancez cet outil.
  3. Définissez le centre de la mise à l'échelle avec la souris ou entrez une coordonnée dans la ligne de commande.
  4. La boîte de dialogue de l'échelle s'affiche et vous permet de saisir le facteur d'échelle.
    Si vous voulez mettre à l'échelle avec deux facteurs différents dans les directions X et Y, décochez le bouton pour la mise à l'échelle proportionnelle :

    Vous pouvez ensuite saisir deux facteurs d'échelle différents pour X et Y. Si vous voulez mettre la sélection à l'échelle à l'aide de la souris, vérifiez le bouton
    du curseur de la souris
  5. Cliquez sur "OK".
  6. Si vous avez déjà choisi de mettre à l'échelle la sélection avec la souris, vous devez maintenant spécifier un point de référence et un point cible pour l'opération de mise à l'échelle.
