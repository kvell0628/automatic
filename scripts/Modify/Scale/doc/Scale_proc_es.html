1. Seleccione las entidades que desea escalar.
  2. Inicie esta herramienta.
  3. Establezca el centro de la escala con el ratón o introduzca una coordenada en la línea de comandos.
  4. Se visualiza el diálogo de la escala, donde se puede introducir el factor de escala.
    Si desea escalar con dos factores diferentes en la dirección X e Y, desmarque el botón para la escala proporcional:

    A continuación, puede introducir dos factores de escala diferentes para X e Y. Si desea escalar la selección con el ratón, compruebe el cursor del ratón
  5. Haga clic en "Aceptar".
  6. Si previamente ha elegido escalar la selección con el ratón, ahora tiene que especificar un punto de referencia y un punto de destino para la operación de escalado.
