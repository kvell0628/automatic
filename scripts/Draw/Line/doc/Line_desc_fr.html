Types de lignes

La plupart des outils de ligne vous permettent de choisir le type de ligne à créer dans la barre d'outils des options. Les types de lignes disponibles sont :

  • Auto:

    Crée automatiquement une ligne du même type qu'une autre ligne choisie. Ceci s'applique à l'outil de ligne parallèle.
  • Segment de ligne :

    Crée des segments de ligne d'un point de départ à un point final.
  • Ligne Infinie :

    Crée des lignes infinies qui passent par deux points donnés. Ce sont des lignes X ou des lignes de construction.
  • Des raies :

    Crée des rayons à partir d'un point de départ donné, à travers un autre point avec une longueur infinie.
