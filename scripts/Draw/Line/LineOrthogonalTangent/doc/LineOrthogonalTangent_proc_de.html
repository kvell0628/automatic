1. Wählen Sie den gewünschten Linientyp in der Optionenwerkzeugleiste.
  2. Wählen Sie die Linie zu welcher die neue Linie orthogonal sein soll.
  3. Wählen Sie den Bogen, Kreis oder die Ellipse zu welcher die Linie tangential sein soll. Es gibt zwei mögliche Lösungen. Wenn Sie die Maus bewegen können Sie jeweils die Vorschau der Tangente sehen, die konstruiert würde, wenn Sie klicken.
