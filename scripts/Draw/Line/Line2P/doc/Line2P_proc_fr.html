1. Sélectionnez le type de ligne souhaité dans la barre d'outils des options.
  2. Spécifiez le point de départ de la première section de ligne. Vous pouvez utiliser la souris ou entrer une coordonnée dans la console.
  3. Spécifiez le point final du premier segment de ligne.
  4. Spécifiez les points d'extrémité des segments de ligne supplémentaires. Cliquez sur le bouton "Fermer" dans la barre d'outils des options pour fermer la séquence :

    Si vous avez besoin d'annuler un seul segment de ligne, vous pouvez le faire en cliquant sur le bouton "Annuler"
  5. Si vous voulez limiter l'angle ou la longueur des segments de ligne, vous pouvez le faire en cliquant sur le bouton "Restreindre l'angle ou la longueur", et en entrant une longueur et/ou un angle.
