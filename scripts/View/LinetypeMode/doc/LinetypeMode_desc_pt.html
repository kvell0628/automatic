Alterna o modo de tipo de linha baseado em tela do desenho atual.
Se este modo estiver ativado, os tipos de linha são otimizados para uma tela de computador. Todas as linhas são exibidas com uma largura de linha baseada na tela (as linhas não ficam mais largas ao ampliar) e todos os padrões são mostrados como padrões fixos, baseados em pixels na tela (traços não ficam mais longos ao aumentar o zoom).
Se este modo estiver desligado, as larguras de linha e os tipos de linha são mostrados em unidades de desenho (padrão).
