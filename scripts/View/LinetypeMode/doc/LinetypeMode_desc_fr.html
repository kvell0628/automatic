Permet de basculer le mode de type de ligne à l'écran du dessin en cours.
Si ce mode est activé, les types de lignes sont optimisés pour un écran d'ordinateur. Toutes les lignes sont affichées avec une largeur de ligne basée sur l'écran (les lignes ne s'élargissent pas lors du zoom avant) et tous les motifs sont affichés à l'écran comme des motifs fixes basés sur les pixels (les tirets ne s'allongent pas lors du zoom avant).
Si ce mode est désactivé, les largeurs de ligne et les types de lignes sont affichés dans les unités de dessin (par défaut).
