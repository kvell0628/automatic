Créez de nouveaux dessins vides à l'aide de cette commande. Les nouveaux dessins sont complètement vides (pas d'entités, un seul calque par défaut, pas de blocs). En tant qu'utilisateur fréquent de QCAD, vous pouvez créer des modèles de dessin avec les calques et les blocs que vous utilisez le plus et probablement une bordure de dessin. Une fois que vous avez un tel modèle, vous pouvez le charger au lieu de commencer un nouveau dessin à partir de zéro.
Veuillez également vous référer à la commande Nouveau à partir du modèle pour apprendre à utiliser les modèles de dessin.
