Questo strumento esporta il disegno corrente in un file PDF. L'esportazione in formato PDF è molto simile alla stampa di un disegno. Molto probabilmente, prima di esportare un disegno in formato PDF, sarà necessario regolare la scala e la posizione del disegno, nonché le impostazioni della pagina. Per questo motivo, QCAD passa automaticamente all'anteprima di stampa quando viene utilizzato questo strumento. Se l'anteprima di stampa non corrisponde alle vostre aspettative, annullate l'esportazione, modificate le impostazioni della pagina e riavviate l'esportazione del PDF.
