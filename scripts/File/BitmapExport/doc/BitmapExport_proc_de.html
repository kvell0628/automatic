In einem Dialog kann die zu erstellende Datei bestimmt werden. Nach der Auswahl des Dateinamens wird ein zweiter Dialog mit Optionen für den Export angezeigt. Hier können Sie die Grösse des Bitmaps in Pixeln bestimmen und zwischen den Hintergrundfarben schwarz oder weiss wählen.
Hinweis: Der Export von Bitmaps mit sehr grosser Grösse kann je nach Hardware sehr lange dauern. Die maximale Grösse für Bitmaps ist Breite x Höhe ≤ 2'147'483'647.
