La commande d'importation insère un dessin à partir d'un fichier sur disque dans le dessin en cours.
Tous les calques utilisés par le dessin importé sont ajoutés à la liste des calques du dessin en cours. Les couches existantes portant le même nom peuvent être écrasées si désiré (option "écraser les couches" dans la barre d'outils des options).
Les références de bloc du dessin importé seront insérées avec les définitions de bloc auxquelles elles se réfèrent. Les blocs du dessin courant peuvent être écrasés si désiré (option "écraser les blocs" dans la barre d'outils des options).
La barre d'outils offre également des outils pour mettre à l'échelle, faire pivoter ou retourner le dessin importé tout en le positionnant.
