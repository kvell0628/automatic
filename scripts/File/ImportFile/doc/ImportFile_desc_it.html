Il comando di importazione inserisce un disegno da un file su disco nel disegno corrente.
Tutti i livelli utilizzati dal disegno importato vengono aggiunti all'elenco dei livelli del disegno corrente. I layer esistenti con lo stesso nome possono essere sovrascritti se si desidera (opzione "Sovrascrivi layer" nella barra degli strumenti delle opzioni).
I riferimenti a blocchi del disegno importato verranno inseriti insieme alle definizioni a blocchi a cui si riferiscono. I blocchi nel disegno corrente possono essere sovrascritti se si desidera (opzione "Sovrascrivi blocchi" nella barra degli strumenti delle opzioni).
La barra degli strumenti delle opzioni offre anche alcuni strumenti per ridimensionare, ruotare o capovolgere il disegno importato durante il posizionamento.
