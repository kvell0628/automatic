El comando de importación inserta un dibujo de un archivo en el disco en el dibujo actual.
Todas las capas utilizadas por el dibujo importado se añaden a la lista de capas del dibujo actual. Las capas existentes con el mismo nombre se pueden sobrescribir si se desea (opción"sobrescribir capas" en la barra de herramientas de opciones).
Las referencias de bloques del dibujo importado se insertarán junto con las definiciones de bloques a las que se refieren. Los bloques del dibujo actual se pueden sobrescribir si se desea (opción"sobrescribir bloques" en la barra de herramientas de opciones).
La barra de herramientas de opciones también ofrece algunas herramientas para escalar, rotar o voltear el dibujo importado mientras se posiciona.
