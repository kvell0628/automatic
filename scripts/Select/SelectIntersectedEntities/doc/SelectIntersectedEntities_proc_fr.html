1. Sélectionnez le mode de sélection dans la barre d'outils des options.
  2. Cliquez sur le point de départ de la ligne d'intersection.
  3. Cliquez sur le point final de la ligne d'intersection.
  4. Toutes les entités intersectées sont sélectionnées ou désélectionnées en fonction du mode de sélection.
